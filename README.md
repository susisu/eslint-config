# @susisu/eslint-config
## Installation
``` shell
# npm
npm i -D @susisu/eslint-config
# yarn
yarn add -D @susisu/eslint-config
```

## Usage
Add `"extends": "@susisu"` to .eslintrc.* file.

``` json
{
  "extends": "@susisu"
}
```
